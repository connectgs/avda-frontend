# AVDA - Arya Vysya Doctors Alliance

## Original Problem Statement
Build a doctors alliance platform for Arya Vysya community members with registration, membership tiers, and directory features.

## Core Requirements
- Doctor registration with multi-step form
- 5 Medical Streams: MBBS, Dental, Ayurveda, Homeopathy, Veterinary
- Membership tiers: Basic (₹100), Premium (₹500), VIP (₹1500)
- Admin approval workflow
- Doctor directory search

## Strategic Direction (First 2-3 Months)
- Primary goal: Maximum registrations
- Secondary goal: Revenue
- Positioning: Community first, revenue second
